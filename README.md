# Sistema de Monitoreo Ambiental

Proyecto desarrollado para la materia **Programación Avanzada**.

## Descripción

Este proyecto tiene como objetivo desarrollar una aplicación para Raspberry Pi capaz de adquirir información de sensores, procesarla y compartirla mediante diferentes mecanismos de comunicación.

A lo largo del curso se irán agregando nuevas funcionalidades hasta obtener un sistema completo basado en Linux y ROS 2.

## Características

- Desarrollo sobre Linux (Ubuntu / WSL)
- Control de versiones con Git
- Organización mediante CMake
- Comunicación con Raspberry Pi
- Lectura de sensores
- Automatización mediante scripts Bash
- Integración con ROS 2

## Requisitos

- Ubuntu 24.04 (WSL2 o instalación nativa)
- Visual Studio Code
- Git
- CMake
- Python 3
- Raspberry Pi (para las prácticas posteriores)
